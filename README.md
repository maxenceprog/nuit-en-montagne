# Nuit en Montagne

## Description

Une petite page web pour donner une vue la plus intuitive possible de l'ensemble des [refuges du FFCAM](https://www.ffcam.fr/rechercher_refuge_chalet.html).

Cette page s'adresse aux gens qui comme moi veulent aller montagne pour le weekend sans avoir une idée précise en tête 
et qui veulent voir en un coup d'oeil les refuges encore disponibles.


## Pour les devs

Ce site est une page static (Github page).

Les données sont mises à jour à 8h du matin et 14h (En heure d'été) via un edit d'un gist contenant les dernieres données scrapées sur le site du FFCAM.
